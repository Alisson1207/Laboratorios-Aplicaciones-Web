## Asincronismo
Los temas realizados en los ejercicios son:

* Código síncrono vs asíncrono
* Promesas
* Async Await

## DOM
* Fundamentos
* Acceder al contenido
* Modificar el contenido
* Modificar el estilo
* Agregar/eliminar elementos
* Eventos
